# On Task Completed Guidelines

Whenever you complete a task or implement a new feature in the Corporate PMO Portal, you must execute the following checklist before wrapping up:

## 1. Visual Verification & Layout Check
- Open the updated or newly created `.html` files in a local web browser.
- Verify that typography defaults to **Montserrat** across the screen.
- Confirm card containers, rounded input fields, pill-shaped buttons, and Iconoir icons align perfectly with the "Pro Max" specifications.
- Check hover interactions and loading animations for fluid, premium transition frames (`0.3s ease`).

## 2. Code Quality & Integration Check
- Format all files using Prettier if available.
- Ensure all variable instantiations in Javascript are encapsulated inside a local closure to prevent namespace leakage.
- Verify that every button and input element has unique, descriptive `id` attributes.
