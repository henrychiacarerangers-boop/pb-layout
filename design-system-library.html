Foundations

The visual language

The website family shares one clear product feel: Montserrat typography, red and navy brand contrast, soft enterprise surfaces, and finance-inspired depth.

Typography

Montserrat

Geometric sans-serif for headings, compact metadata, labels, and operational UI.

Color

red + navy + support gray

Red carries urgency and primary action. Navy anchors structure and trust.

Surface

white cards + soft shadow

Most content is grouped through clean cards, subtle borders, and restrained elevation.

Motion

hover lift + soft transitions

Interactions are quiet: tab movement, card lift, carousel fade, toast entry, and ticker flow.

Governance

How to mature the system

The site already has a strong design language. The next step is consolidation: fewer names, clearer variants, and one token source of truth.

Recommended next move

Merge duplicated CSS across PMO, PMO corporate, UTC, and Corporate Website. Then formalize repeated classes like badge-alert, view-tab, gateway-tab-btn, and custom-table as official library components.
